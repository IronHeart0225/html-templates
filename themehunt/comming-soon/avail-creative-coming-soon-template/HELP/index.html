Avail is a HTML5 coming soon template. It offers simple design with unique transition animation. Avail was built with latest Bootstrap framework so this template will look great on any devices. You could use Avail for any type of your project. Its easy to use and well documented. You could contact me anytime from Avail support tab if you need my help or have some question about this template. Avail comes with various features, you could check the main features below.

Thank you for purchasing my theme. If you have any questions that are beyond the scope of this help file, please feel free to email via my user page contact form here. Thanks so much!

Features

  • Countdown
  • Slideshow Background
  • 5 Homepage Background Variants
  • Fresh, Simple and Clean Design
  • Valid HTML5 and CSS3
  • Built with latest Bootstrap framework
  • Page Transition
  • Single Image Background + Star Effect (Constellation)
  • YouTube Video Background
  • Carousel Slider
  • Fully Responsive Design
  • Retina Ready
  • CSS3 Animation
  • Font Awesome + Simple Line Icon Integration (300+ icons)
  • Working AJAX subscription form with Mailchimp integration
  • Working AJAX contact form
  • Google Map Integration
