HTML Structure

All of the information within the main content is nested within multiple section with tag of "section". Here is the general structure. All style.css are put in head tag, put all js files(In addition to jquery.js) before </body> tag.

CSS Files and Structure

There are 3 main CSS files in this theme.

If you would like to edit a specific section of the site, simply find the Businessropriate label in the CSS file, and then scroll down until you find the Businessropriate style that needs to be edited.

Open the css folder from this location 'assets/css', you'll find all css files.

  • main.css - theme common styles
  • bootstrap.min.css - Twitter bootstrap styles
  • font-awesome.min.css - Font Awesome stylesheet

Note:

Make sure the main.css is below other css files without alternative color style

 

Logo & Menu Settings

Change your logo text and link navbar markup:

    <!-- logo -->
    <a class="brand-logo" href="#Sliders"><img src="img/logo.png" alt="Karbar"> </a>

The Image logo size is limited max height as 500px.

MENU:

Edit menu here:
