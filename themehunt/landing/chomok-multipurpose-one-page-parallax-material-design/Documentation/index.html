Main File Structure

All the files are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "Chomok - Main File" folder.
You will find your packege, then choose index.html or othre versions.
Also you will see 5 directories, css, js, fonts, images, ajaxform.
In css folder Content you will find style.css and other styling files, all the other files expect the htmls.
Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts. 


    main-theme-html/
	├── ajaxform
    ├── css
    ├── fonts
    ├── images
    ├── js
    ├── index.html
    ├── style.css
