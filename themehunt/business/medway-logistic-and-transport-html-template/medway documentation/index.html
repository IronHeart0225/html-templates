Getting Started

MedWay is a HTML/CSS template based on Material Design for Bootstrap 4 framework. It allows to built impressive, better & faster, mobile friendly websites.

HTML Structure

This is the base of each HTML file:


                <!-- header-section - start
                ================================================== -->
                <header id="header-section" class="header-section">
                ...
                </header>
                <!-- header-section - end
                ================================================== -->


                <!-- your-class-section - start
                ================================================== -->
                <section id="your-id-section" class="your-class-section">
                ...
                </section>
                <!-- your-class-section - end
                ================================================== -->
                
                <!-- footer-section - start
                ================================================== -->
                <footer id="footer-section" class="footer-section">
                ...
                </footer>
                <!-- footer-section - end
                ================================================== -->

Grid System

MedWay Template supporst all features of Bootstrap / Material Design for Bootstrap 4 Grid System - go to Bootstrap / Material Design for Bootstrap 4 and check how does it work.

Configuration

The template includes some configurable elements - here you will find informations how to use them!

Slider / carousel Setting

There are two slider used in MedWay "slick slider and owl carousel 2". Go to the slick slider / owl carousel 2 and check how does it work. 



                <!-- slick - start -->
                <div class="your-class">
                  <div></div>
                  <div></div>
                  <div></div>
                </div>
                <!-- slick - end -->


                <!-- owl carousel 2 - start -->
                <div class="your-class owl-carousel owl-theme">
                  <div class="item"></div>
                  <div class="item"></div>
                  <div class="item"></div>
                </div>
                <!-- owl carousel 2 - end -->
