Instalation

Follow the steps below to get started with your Site Template:

  1. Open the Package/HTML Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:


    • HTML/css - Extra Stylesheets Folder
    • HTML/images - Images Folder
    • HTML/js - Javacripts Folder
    • HTML/style.css - Main Stylesheet File
    • HTML/index.html - Index File/Homepage



    4. The other files can be used according to your preferences. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style

    Note:


    Doockers template have 8 home pages you chose your home according to your website and business...

Smooth Scrolling

Doockers Multipurpose HTML5 Template enables Smooth Scrolling on Windows Desktop PCs for a Smooth experience. You can enables it by simply adding the .wow and .fadeIn(transition class name) in any tag.


Page Loading Transitions

You can show Interactive loaders to your Visitors while the Pages of you Website loads in the background & then Reveal your Pages with CSS3 Transitions. Page Loading Transitions are enabled by default. To disable the Transition, you can simply remove the .loader

Changing Fonts

Change your Fonts on the Fly as we have included a css/style.css file to manage the custom fonts you include with ease. By default, Doockers Template uses 1 Fonts namely: Roboto from the Google Fonts Library. You can find the Linking to the Font Files in the head tag of all the .html files.

 <link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">

In order to change the Fonts, you will need to Edit the Above Links with your Custom Font if you plan to use a Google Font or Remove it complete if you plan to use a Self Hosted font. Here is an Example for using Self Hosted Fonts.

What's included

Once you download the template, you can see a folder with several files and sub folders. You can read description of some important files in continue.

    • css
      • master.css All files import in this master.css file
      • style.css All custom styles in this file
      • Bootstrap.min.css Minified version of all required bootstrap files
      • font-awesome.min.css Minified version of all required font-awesome / Glyphicons files
      • color.css Minified version of all required all color property
      • responsive.css Minified version of all required reponsive files
    • fonts
      • FontAwesome fontawesome-webfont
      • glyphicons glyphicons-halflings-regular
      • Flat-UI-Icons.dev for icons
      • roboto roboto-regular/bold/light-webfont
    • Images
      • JPG / PNG All Images demo images and replace all images with your original images
    • js
