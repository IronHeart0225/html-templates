Slider

  • 1. Open the file "home2.html" / "onepage2.html" / "boxed2.html"
  • 2. Find the following code 
       
                            <div id="tarikzhero" class="owl-carousel owl-theme">
                                ...
                            <div/>
    this code holds the slider/carousel
  • 3. within the code above, there are divs with class name- 'item'. these divs hold codes for individual slides. Inside each div with 'item' class there is a div with class- 'slide one' / 'slide two' etc. This class has the slide image set as background.
  • 4. to change the images go to 'style.css' file in css folder and find the following code- 
    
                            .slide.one {
                                background: url(../images/slider/slide-1.jpg);
                            }

                            .slide.two {
                                background: url(../images/slider/slide-2.jpg);
                            }
    now change the image refernce as you wish.

Animated Typed Text

  • 1. Open the file "home.html" / "onepage.html" / "boxed.html"
  • 2. Find the following code 
       
                            <section class="background-area img-bg">
                                ...
                            <section/>
    this code holds the image background
  • 3. notice that in the above code we have a class - 'img-bg' for the section tag. This class has our image set as background.
  • 4. to change the image go to 'style.css' file in css folder and find the following code- 
    
                            .img-bg {
                                background: url(../images/bg/hero.jpg) 48% 48%;
                                background-size: cover;
                            }
    now change the image refernce as you wish.

Navigation

                                <li><a href="team.html">team</a></li>

                            <div class="main-menu">
